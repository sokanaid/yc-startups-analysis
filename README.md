# YC Startups Analysis — Solo Developer Edition

Анализ стартапов Y Combinator за последние 2 года (S23, W24, S24, W25) с фокусом на идеи для соло-разработчика с минимальными вложениями.

## Файлы

| Файл | Описание |
|------|----------|
| `yc_startups_analysis.xlsx` | Основной файл анализа с 4 листами |

## Структура Excel-файла

### 1. База данных
40 стартапов из батчей S23 → W25 с полными данными:
- Батч, сектор, описание, статус
- Количество основателей, наличие технического CTO
- **Solo-оценка (1–5)** — насколько реалистично сделать в одиночку
- Трафик сайта, инвестиции, конкуренты
- Идея для вдохновения на основе каждого стартапа

### 2. Топ для соло
12 лучших стартапов с Solo-Score = 4–5, отфильтрованных вручную.
Включает стек, монетизацию и конкретную адаптацию идеи для тебя.

### 3. Аналитика
- Распределение по секторам (pie chart)
- Рост доли AI по батчам S23→W25 (bar chart)
- Топ-10 по трафику (bar chart)
- Solo-friendliness score distribution

### 4. Идеи для тебя
**10 конкретных идей**, вдохновлённых реальными YC-стартапами:

| # | Идея | Недели до MVP |
|---|------|---------------|
| 1 | AI-copilot для конкретной профессии | 3–4 нед |
| 2 | Voice AI агент для локального бизнеса | 2–3 нед |
| 3 | UI-компонент кит для нишевого фреймворка | 6–8 нед |
| 4 | No-code AI workflow для 1 индустрии | 8–10 нед |
| 5 | AI браузер-автоматизация для конкретного сценария | 2–4 нед |
| 6 | AI исследовательский ассистент для вертикали | 4–6 нед |
| 7 | Open-source auth lib для нишевой экосистемы | 4–6 нед |
| 8 | AI role-play тренажёр для профессии | 3–5 нед |
| 9 | AI обработка документов конкретного типа | 4–6 нед |
| 10 | Аналитика/дашборд SaaS для конкретной роли | 6–8 нед |

## Ключевые выводы

- **67% стартапов S24 и 72% стартапов W25** — AI-компании (2 года назад было 15%)
- **Solo-friendly стартапы** — чаще всего Developer Tools и нишевые AI-инструменты
- Самые быстрорастущие паттерны: **open-source → hosted SaaS** и **API product on top of infrastructure**
- YC всё больше принимает соло-основателей: в W25 ~18% компаний с одним основателем
- Better Auth (W25) и StarSling (S25) — примеры успешных соло-основателей в YC 2025
