# YC Startups Analysis — Solo Developer Edition

Анализ стартапов Y Combinator за последние 2 года (S23, W24, S24, W25) с фокусом на идеи для соло-разработчика с минимальными вложениями.

**Включает честный анализ рисков AI-обёрток** — почему 80-95% из них умирают и какие паттерны реально работают.

## Структура Excel-файла

### 1. База данных
40 стартапов из батчей S23 → W25 с полными данными:
- Батч, сектор, описание, статус
- Количество основателей, наличие технического CTO
- **Solo-оценка (1–5)** — насколько реалистично сделать в одиночку
- Трафик сайта, инвестиции, конкуренты
- Идея для вдохновения на основе каждого стартапа

### 2. Топ для соло
12 лучших стартапов с Solo-Score = 4–5, отфильтрованных вручную.
Включает стек, монетизацию и конкретную адаптацию идеи.

### 3. Аналитика
- Распределение по секторам (pie chart)
- Рост доли AI по батчам S23→W25 (bar chart)
- Топ-10 по трафику (bar chart)
- Solo-friendliness score distribution

### 4. Идеи для тебя (обновлено с учётом рисков)

Идеи разделены на 3 категории:

| Категория | Идеи |
|-----------|------|
| **SAFE** — не обёртки, OpenAI не конкурент | Open-source dev tool, Нишевый SaaS (AI = фича), Микро-утилита |
| **RISKY** — обёртки с возможной защитой | AI-тренажёр переговоров, AI-copilot для 1 профессии |
| **AVOID** — высокий риск смерти от платформ | Voice AI агент, AI research assistant, Browser automation, AI document parser |

### 5. Риски AI-обёрток (NEW)
- Кладбище: Jasper (-54% revenue), Builder.ai (банкрот), Phind (закрыт через месяц после $10M раунда)
- Статистика: 80-95% обёрток умирают, только 2-5% доходят до $10K/мес
- Выжившие и почему: Harvey AI ($5B), Copy.ai (pivot), FormulaBot (микро-ниша)
- Паттерн: что умирает vs что выживает

## Ключевые выводы

### Рынок
- **67% стартапов S24 и 72% W25** — AI-компании (2 года назад было 15%)
- YC всё больше принимает соло-основателей: в W25 ~18% компаний с одним основателем
- Better Auth (W25) и StarSling (S25) — примеры успешных соло-основателей в YC

### Риски AI-обёрток (честный анализ)
- **80-95% AI-обёрток** умирают, не заработав денег
- **41% YC AI-стартапов** — обёртки с низкой ценностью (Stanford)
- Конкурентное преимущество живёт **3-6 месяцев** — потом копируют
- Jasper ($131M funding) потерял 54% revenue когда вышел ChatGPT
- Phind закрылся через месяц после Series A

### Что работает для соло-разработчика
- **Open-source dev tools** — community moat, OpenAI не конкурирует
- **Нишевый SaaS где AI = фича** — дашборд/CRM для конкретной профессии
- **Микро-утилиты а ля FormulaBot** — узкая задача, $500K ARR с 87% маржой
- **Ключевой принцип:** AI как инструмент внутри продукта ≠ AI как продукт
